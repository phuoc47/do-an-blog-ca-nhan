Những tính năng mới trong ES6 JavaScript

  • bởi admin
  • 20/10/2025
Giới thiệu

ECMAScript 2015 (hay ES6) là một bước ngoặt quan trọng của JavaScript, mang lại hàng loạt cú pháp và tính năng hiện đại giúp code dễ đọc, ngắn gọn, và mạnh mẽ hơn. Việc hiểu và áp dụng ES6 là yêu cầu cơ bản cho mọi lập trình viên web hiện nay.

1. let và const – Khai báo biến hiện đại

Trước ES6, JavaScript chỉ có từ khóa var – dễ gây lỗi vì phạm vi biến (scope) không rõ ràng. Nay bạn nên dùng let và const:

let count = 10;     // có thể thay đổi
const pi = 3.14;    // không thể gán lại

if (true) {
    let x = 5;
    console.log(x); // ✅ 5
}
// console.log(x); ❌ Lỗi – ngoài phạm vi

Lưu ý: Dùng const cho giá trị không đổi, let cho biến thay đổi. Hạn chế dùng var.

2. Arrow Function – Hàm ngắn gọn

Hàm mũi tên giúp cú pháp ngắn và tự động “bind” ngữ cảnh this:

// Trước đây
function sayHello(name) {
    return "Xin chào, " + name;
}

// ES6
const sayHello = name => `Xin chào, ${name}`;
console.log(sayHello("Phước"));
3. Template String – Chuỗi có biến nội suy

Thay vì phải nối chuỗi rườm rà, ES6 cho phép dùng dấu ` để viết chuỗi dễ hơn:

const ten = "Phước";
const tuoi = 21;
console.log(`Tôi là ${ten}, năm nay ${tuoi} tuổi.`);
4. Destructuring – Rút gọn khi trích xuất dữ liệu

Thay vì truy cập từng thuộc tính, bạn có thể “bóc tách” trực tiếp:

const nguoi = { ten: "Alex", tuoi: 25, diachi: "TP.HCM" };
const { ten, tuoi } = nguoi;
console.log(ten, tuoi);
5. Spread & Rest – Làm việc với mảng và tham số linh hoạt

Spread (...): Dùng để sao chép hoặc gộp mảng.

const a = [1, 2, 3];
const b = [...a, 4, 5];
console.log(b); // [1, 2, 3, 4, 5]

Rest (...): Gom nhiều tham số thành một mảng.

function tong(...nums) {
    return nums.reduce((a, b) => a + b, 0);
}
console.log(tong(1, 2, 3, 4)); // 10
6. Classes – Lập trình hướng đối tượng trong JavaScript

ES6 hỗ trợ cú pháp class tương tự Java hoặc C#, giúp code dễ hiểu hơn:

class Nguoi {
    constructor(ten, tuoi) {
        this.ten = ten;
        this.tuoi = tuoi;
    }
    chao() {
        console.log(`Xin chào, tôi là ${this.ten}`);
    }
}
const phuoc = new Nguoi("Phước", 21);
phuoc.chao();
7. Promise – Giải pháp xử lý bất đồng bộ

Thay vì callback lồng nhau (“callback hell”), ES6 mang đến Promise – giúp xử lý tuần tự, rõ ràng hơn:

const layDuLieu = new Promise((resolve, reject) => {
    setTimeout(() => resolve("Dữ liệu đã tải xong!"), 2000);
});

layDuLieu.then(res => console.log(res));
8. Default Parameters – Giá trị mặc định cho hàm
function greet(name = "Khách") {
    console.log(`Xin chào, ${name}!`);
}
greet(); // Xin chào, Khách!
9. Kết luận

ES6 không chỉ cải thiện hiệu suất, mà còn làm JavaScript trở nên “hiện đại” hơn, gần gũi với các ngôn ngữ khác. Hãy luyện tập viết lại code cũ bằng cú pháp ES6 – bạn sẽ thấy sự khác biệt lớn trong tốc độ và trải nghiệm lập trình.

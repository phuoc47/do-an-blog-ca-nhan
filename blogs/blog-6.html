Thao tác DOM với JavaScript

  • bởi admin
  • 17/10/2025
1. Giới thiệu về DOM

DOM (Document Object Model) là mô hình đối tượng tài liệu mà trình duyệt tạo ra khi tải một trang web. Nó biểu diễn toàn bộ HTML dưới dạng cây (tree structure), cho phép JavaScript truy cập, thay đổi hoặc xóa các phần tử.

Hiểu đơn giản: HTML là nội dung tĩnh, còn JavaScript thông qua DOM giúp trang web trở nên “sống động”.

2. Chọn phần tử DOM

Có nhiều cách để chọn phần tử HTML bằng JavaScript:

// Cách 1: Chọn theo ID
const title = document.getElementById("title");

// Cách 2: Chọn theo class
const items = document.getElementsByClassName("item");

// Cách 3: Chọn theo tag
const divs = document.getElementsByTagName("div");

// Cách 4: Sử dụng querySelector hiện đại
const button = document.querySelector(".btn-primary");
const allButtons = document.querySelectorAll(".btn");

Gợi ý: Nên dùng querySelector và querySelectorAll vì chúng linh hoạt và hiện đại hơn.

3. Thay đổi nội dung và thuộc tính

JavaScript có thể cập nhật text, HTML hoặc thuộc tính của phần tử:

// Thay đổi nội dung văn bản
document.getElementById("title").textContent = "Xin chào DOM!";

// Thay đổi nội dung HTML
document.getElementById("content").innerHTML = "<b>Đây là đoạn văn mới!</b>";

// Thay đổi thuộc tính
document.querySelector("img").src = "images/new-photo.jpg";
4. Tạo và xóa phần tử động

DOM cho phép tạo thêm các phần tử mới mà không cần reload trang:

const newItem = document.createElement("li");
newItem.textContent = "Phần tử mới";
document.querySelector("ul").appendChild(newItem);

// Xóa phần tử
const itemToRemove = document.querySelector("li:first-child");
itemToRemove.remove();
5. Thao tác với CSS bằng DOM

Bạn có thể thay đổi giao diện thông qua thuộc tính style hoặc thêm/bớt class:

const box = document.querySelector(".box");
box.style.backgroundColor = "lightblue";
box.style.padding = "10px";

// Thêm / Xóa class
box.classList.add("active");
box.classList.remove("hidden");
6. Lắng nghe sự kiện (Event Listener)

Sự kiện (event) giúp người dùng tương tác với trang, ví dụ khi click chuột hoặc nhập dữ liệu:

const btn = document.querySelector("#clickMe");
btn.addEventListener("click", () => {
    alert("Bạn vừa click vào nút!");
});

Có thể lắng nghe nhiều loại sự kiện khác nhau như mouseover, keydown, submit,…

7. Kết hợp DOM và Logic động

Bạn có thể dùng DOM để tạo hiệu ứng nhỏ hoặc chức năng tương tác. Ví dụ: đổi màu nền ngẫu nhiên khi click:

const btn = document.querySelector("#colorBtn");
btn.addEventListener("click", () => {
    const color = "#" + Math.floor(Math.random() * 16777215).toString(16);
    document.body.style.backgroundColor = color;
});
8. Tổng kết

DOM là nền tảng quan trọng của lập trình web. Thành thạo thao tác DOM giúp bạn:

  • Tạo trang web tương tác, sinh động.
  • Xây dựng hiệu ứng hoặc component UI.
  • Hiểu sâu hơn khi học framework như React, Vue, hay Angular.

Lời khuyên: Hãy thực hành thật nhiều với các bài nhỏ như: bật/tắt đèn, đếm số lần click, thêm phần tử vào danh sách. Việc nắm vững DOM sẽ giúp bạn tiến xa hơn rất nhiều trong JavaScript.
